## Software-Architektur & Java

Die Software-Architektur in Java entwickelt sich weiter, insbesondere durch die Einführung der JEP 533, die das Exception Handling in der strukturierten Nebenläufigkeit verbessert. Zudem wird die Foreign Function & Memory API hervorgehoben, die den Zugriff auf C-Libraries erleichtert.

- JEP 533 führt einen neuen ExecutionException-Typ ein und aktualisiert das Joiner-Interface.
- Die Foreign Function & Memory API ermöglicht einen einfacheren Zugriff auf C-Funktionen als das veraltete JNI.
- Paulo Arruda präsentiert Fortschritte bei der Entwicklung spezialisierter Agenten in der KI bei Shopify.
